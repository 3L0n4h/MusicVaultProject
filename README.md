# Music Vault Project
Music Vault is a web application built using Springboot, hibernate, thymeleaf, and sqlexpress 2016. It features add, edit, delete for songs. artists and shows. It has voting system for top 10 songs for the month. 

## Programming Language Details
* Java
* Springboot Framework and Thymeleaf
* CSS Template Templatemo and Bootstrap 5.3.3
* SQL Scripts

## Developer Tools 
* Eclipse IDE 2022-03 4.23.0
* Java version 17.0.4.1
* Spring Framework 3.4.10
* MS SQL Server 2016 or SQL Express 13.0.6404.1
* Git version 2.34.1.windows.1
